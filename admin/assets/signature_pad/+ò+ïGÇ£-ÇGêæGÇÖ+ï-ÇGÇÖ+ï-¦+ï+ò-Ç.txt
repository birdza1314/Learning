ทำแบบฟอร์มที่บันทึกลายเซ็นได้ (Signature Pad)

*วิธีการติดตั้ง*

1. ลองดูตัวอย่างการนำไปใช้ที่ไฟล์ example.php ครับ

2. ให้ทำการ Set Permission โฟลเดอร์ "doc_signs" เป็น 777 ด้วย

3. ก็สามารถนำไปพัฒนาต่อได้เลยครับ ไม่ยากๆ ลองดู Code ในหน้าตัวอย่างดูก่อนครับ

4. แค่นี้ครับ
